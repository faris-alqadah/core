Retains the top K clusters by checking for overlapping clusters then keeping the higher quality cluster if overlap occurs. 

Author: Faris Alqadah Helper functions for keeping track of the top K clusters during an enumeration algorithm

Attempts to add the cluster c to v, this can happen in three cases v.size() $<$ k OR there does not exist any cluster x in v s.t Ovlp(c,x) $>$ ovlpThresh \&\& Quality(c) $>$ min(Quality(x)) for any x in v OR there exists at least once cluster c s.t. ovlp(c,x) $>$ ovlpThresh \&\& Quality(c) $>$ Quality(x) then c replaces x
